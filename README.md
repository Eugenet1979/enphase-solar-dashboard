# ☀️ Enphase Solar Dashboard Pro for Home Assistant

A modern, responsive Home Assistant dashboard for Enphase Envoy systems with individual microinverter monitoring.

> Designed for Home Assistant 2026+ using the official Enphase integration.

---

## Features

- ⚡ Live solar production
- 🏠 Live home consumption
- 🌐 Grid import/export
- ☀️ Animated power flow
- 🔋 Battery-ready (optional)
- 📈 Production history
- 📊 Daily, weekly, monthly and yearly analytics
- 🟢 Individual microinverter monitoring
- 🔥 Color-coded array heat map
- 🚨 Offline inverter detection
- ⚠️ Low-production panel detection
- 📱 Responsive mobile layout
- 🖥️ Desktop dashboard
- 🌙 Dark mode optimized
- 🌤️ Solcast ready

---

## Dashboard Views

### Overview

- Current Solar Production
- Home Consumption
- Grid Import/Export
- Today's Production
- Today's Consumption
- Lifetime Production
- Animated Power Flow

---

### Array View

Interactive roof layout showing:

- Every microinverter
- Live wattage
- Production color
- Offline indicators
- History popup

---

### Analytics

- 24 Hour Graph
- 7 Day Production
- Monthly Production
- Yearly Production
- Lifetime Energy
- Efficiency Metrics

---

### Diagnostics

Automatically detects

- Weak panels
- Offline inverters
- Lowest producer
- Highest producer
- Production imbalance
